#  Gaming Behavior & Mental Health Analytics

An end-to-end data analytics project analyzing the impact of gaming habits on mental health using **10 Million rows** of data.

##  Project Overview
This project identifies patterns in gaming behavior that correlate with depression risk. I used a hybrid approach of **Big Data processing** (Python), **Machine Learning** (K-Means & Random Forest), and **Business Intelligence** (Power BI).

## Tech Stack
- **Data Processing:** Polars, Pandas
- **Machine Learning:** Scikit-Learn (Random Forest, K-Means Clustering)
- **Visualization:** Power BI, Matplotlib, Seaborn

##  Key Insights
- **User Segmentation:** Identified 3 distinct gamer personas: *Casual*, *Hardcore*, and *Fitness-Conscious*.
- **Correlation:** Sleep quality was found to be a stronger predictor of depression risk than gaming hours alone.
- **Risk Prediction:** Built a model to identify "At-Risk" users based on behavioral data.
